RLHF Pipeline

Three-phase training pipeline — SFT → Reward → PPO.

📦
Phase 1

SFT Fine-Tuning

GPT-2 fine-tuned on cover letter corpus with LoRA for memory efficiency.

python scripts/train_sft.py
→
🏆
Phase 2

Reward Model

DistilBERT trained on pairwise preference data with margin ranking loss.

python scripts/train_reward.py
→
🔄
Phase 3

PPO RLHF Loop

Policy optimised with PPO + KL penalty. Reward model scores each rollout.

python scripts/train_ppo.py
→
